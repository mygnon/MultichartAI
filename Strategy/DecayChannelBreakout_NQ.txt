{===============================================================================
  DecayChannelBreakout_NQ
  -------------------------------------------------------------------------------
  PowerLanguage (EasyLanguage / MultiCharts64) trading strategy for CME.NQ futures.
  Long + Short. DECAYING-EXTREME channel breakout — a Donchian whose old extremes
  FADE toward price with age — entered intrabar via STOP + ReentryBars cooldown
  + ATR chandelier trail.

  Futures port of DecayChannelBreakout_crypto: the _Crypto1MUSD ~$1M-notional position
  sizing has been REMOVED -- orders use the strategy's default sizing (set in
  Format Signals -> Properties -> default contracts).

  WHY A NEW BREAKOUT REFERENCE:
  The OOS-robust breakout engine is the same skeleton everywhere — a rolling/non-lagged
  reference LEVEL + an intrabar STOP fill AT that level + a WIDE ATR(14) chandelier
  trailing exit + a post-exit ReentryBars cooldown. Seventeen references varied the
  field / lag / anchor / volume-weighting / band-width / composition of the level.
  This probes an 18th reference on a NEW AXIS — REFERENCE MEMORY DECAY: how fast
  should the breakout level FORGET old extremes? Donchian never forgets (within the
  window); here every extreme is discounted linearly by its age:
      disc(i)  = BandMult * ATR * (i-1) / Length          { 0 for the newest bar }
      UpDecay  = Max over i=1..Length of ( High[i] - disc(i) )
      DnDecay  = Min over i=1..Length of ( Low[i]  + disc(i) )
      long STOP at UpDecay ; short STOP at DnDecay
  A freshly-printed high demands a full break; a Length-bar-old high has faded by up
  to BandMult*ATR — stale resistance decays, so entries on aged levels come EARLIER
  while fresh extremes still gate hard. BandMult = 0 reduces EXACTLY to Donchian
  (clean nested A/B); larger BandMult = faster-fading memory. Ties to the lag law:
  fading memory is a softer, earlier level — does it beat the hard extreme, or is
  Donchian's total recall part of the edge?

  This is NOT mean-reversion: it BUYS the upper break and SHORTS the lower break
  (continuation), with the wide ATR trail letting winners run.

    - Long  : flat & cooldown done -> Buy       next bar at UpDecay STOP
    - Short : flat & cooldown done -> SellShort  next bar at DnDecay STOP
    - Exit  : ATR(14) chandelier trailing stop (the decisive exit). Flat-only.

  Inputs (all optimizable; names match the optimizer's ParamAxis exactly):
    Length      - channel lookback, bars (completed bars 1..Length)
    BandMult    - TOTAL decay across the window in ATR(14) multiples
                  (0 = pure Donchian; per-bar fade = BandMult*ATR/Length)
    ATRMult     - trailing-stop distance in ATR(14) multiples (wider = looser)
    ReentryBars - bars to stay flat after an exit before a new entry is allowed

  ⚠️ Levels are computed from COMPLETED bars (indices [1..Length]) — no extra [1]
     shift needed. O(Length) loop per bar; keep Length <= 300.

  Defaults: Length=20, BandMult=1.0, ATRMult=3.0, ReentryBars=0
  Author: generated for Tim / MultichartAI, 2026-07-07
===============================================================================}

inputs:
    Length( 20 ),
    BandMult( 1.0 ),
    ATRMult( 3.0 ),
    ReentryBars( 0 ) ;

variables:
    ATRv( 0 ),
    ii( 0 ),
    disc( 0 ),
    UpDecay( 0 ),
    DnDecay( 0 ),
    flatBars( 0 ),
    TrailLong( 0 ),
    TrailShort( 0 ) ;

ATRv = AvgTrueRange( 14 ) ;

{ ---- decaying-extreme channel of the COMPLETED bars (age fades old extremes) ---- }
UpDecay = High[1] ;
DnDecay = Low[1] ;
for ii = 2 to Length
begin
    disc = BandMult * ATRv * ( ii - 1 ) / Length ;
    if High[ii] - disc > UpDecay then
        UpDecay = High[ii] - disc ;
    if Low[ii] + disc < DnDecay then
        DnDecay = Low[ii] + disc ;
end ;

{ ---- flat-cooldown counter: consecutive bars with no open position ---- }
if MarketPosition = 0 then
    flatBars = flatBars + 1
else
    flatBars = 0 ;

{ ---- entries: decay-channel breakout STOP, only when flat + cooldown done ---- }
if MarketPosition = 0 and flatBars >= ReentryBars then
begin
    Buy ( "DyLE" ) next bar at UpDecay stop ;
    SellShort ( "DySE" ) next bar at DnDecay stop ;
end ;

{ ---- ATR chandelier trailing stop: long ---- }
if MarketPosition = 1 then
begin
    if BarsSinceEntry = 0 then
        TrailLong = EntryPrice - ATRMult * ATRv
    else
        TrailLong = MaxList( TrailLong, Close - ATRMult * ATRv ) ;
    Sell ( "DyLX" ) next bar at TrailLong stop ;
end ;

{ ---- ATR chandelier trailing stop: short ---- }
if MarketPosition = -1 then
begin
    if BarsSinceEntry = 0 then
        TrailShort = EntryPrice + ATRMult * ATRv
    else
        TrailShort = MinList( TrailShort, Close + ATRMult * ATRv ) ;
    BuyToCover ( "DySX" ) next bar at TrailShort stop ;
end ;
