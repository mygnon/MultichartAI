{===============================================================================
  OpenRangeBreakout_NQ
  -------------------------------------------------------------------------------
  PowerLanguage (EasyLanguage / MultiCharts64) trading strategy for CME.NQ futures.
  Long + Short. TODAY'S-OPEN +/- average-day-range breakout (classic Larry-Williams
  volatility breakout) entered intrabar via STOP + ReentryBars cooldown + ATR
  chandelier trail.

  Futures port of OpenRangeBreakout_crypto: the _Crypto1MUSD ~$1M-notional position
  sizing has been REMOVED -- orders use the strategy's default sizing (set in
  Format Signals -> Properties -> default contracts).

  WHY A NEW BREAKOUT REFERENCE:
  The OOS-robust breakout engine is the same skeleton everywhere — a rolling/non-lagged
  reference LEVEL + an intrabar STOP fill AT that level + a WIDE ATR(14) chandelier
  trailing exit + a post-exit ReentryBars cooldown. The SESSION-ANCHORED family so far:
    - PivotBreakout      : levels derived from the PRIOR day's H/L/C (pivot +/- k*range)
    - DayChannelBreakout : extremes of the last N COMPLETED days
    - WeekChannelBreakout: extremes of the last N COMPLETED weeks
  This probes a 15th reference — the third ANCHOR type: TODAY'S SESSION OPEN. The
  classic volatility breakout,
      AvgRng = Average of (HighD(i)-LowD(i)) over the last Length completed days
      UpLvl  = OpenD(0) + BandMult * AvgRng
      DnLvl  = OpenD(0) - BandMult * AvgRng
  The anchor RESETS every day at the session open — each day is a fresh bet that a
  move of BandMult x typical-day-range away from the open continues (range-expansion
  logic), vs Pivot/DayChannel whose levels carry over from completed sessions.
  Levels are CONSTANT within the day. Completes the anchoring-axis sample set:
  prior-day derived / multi-day extremes / current-day open.

  This is NOT mean-reversion: it BUYS the upper break and SHORTS the lower break
  (continuation), with the wide ATR trail letting winners run.

    - Long  : flat & cooldown done -> Buy       next bar at UpLvl STOP
    - Short : flat & cooldown done -> SellShort  next bar at DnLvl STOP
    - Exit  : ATR(14) chandelier trailing stop (the decisive exit). Flat-only.

  Inputs (all optimizable; names match the optimizer's ParamAxis exactly):
    Length      - completed-day lookback for the average day range, DAYS
    BandMult    - breakout distance from today's open in average-day-range multiples
                  (classic LW ~0.5-1.0)
    ATRMult     - trailing-stop distance in ATR(14) multiples (wider = looser)
    ReentryBars - bars to stay flat after an exit before a new entry is allowed

  ⚠️ Requires an INTRADAY chart (OpenD/HighD/LowD, same family as PivotBreakout /
     DayChannel). Average range computed with an explicit HighD(i)/LowD(i) loop.
     Keep Length <= 50. On futures the session = the exchange session of the chart.

  Defaults: Length=10, BandMult=0.7, ATRMult=3.0, ReentryBars=0
  Author: generated for Tim / MultichartAI, 2026-07-06
===============================================================================}

inputs:
    Length( 10 ),
    BandMult( 0.7 ),
    ATRMult( 3.0 ),
    ReentryBars( 0 ) ;

variables:
    ATRv( 0 ),
    ii( 0 ),
    RngSum( 0 ),
    AvgRng( 0 ),
    UpLvl( 0 ),
    DnLvl( 0 ),
    flatBars( 0 ),
    TrailLong( 0 ),
    TrailShort( 0 ) ;

ATRv = AvgTrueRange( 14 ) ;

{ ---- average day range of the last Length COMPLETED days ---- }
RngSum = 0 ;
for ii = 1 to Length
begin
    if HighD( ii ) > 0 and LowD( ii ) > 0 then
        RngSum = RngSum + ( HighD( ii ) - LowD( ii ) ) ;
end ;
AvgRng = RngSum / MaxList( Length, 1 ) ;

{ ---- today's-open volatility-breakout levels (reset each session; intra-day constant) ---- }
UpLvl = OpenD( 0 ) + BandMult * AvgRng ;   { long breakout level }
DnLvl = OpenD( 0 ) - BandMult * AvgRng ;   { short breakout level }

{ ---- flat-cooldown counter: consecutive bars with no open position ---- }
if MarketPosition = 0 then
    flatBars = flatBars + 1
else
    flatBars = 0 ;

{ ---- entries: open-range breakout STOP, only when flat + cooldown done ---- }
if MarketPosition = 0 and flatBars >= ReentryBars and AvgRng > 0 then
begin
    Buy ( "OrLE" ) next bar at UpLvl stop ;
    SellShort ( "OrSE" ) next bar at DnLvl stop ;
end ;

{ ---- ATR chandelier trailing stop: long ---- }
if MarketPosition = 1 then
begin
    if BarsSinceEntry = 0 then
        TrailLong = EntryPrice - ATRMult * ATRv
    else
        TrailLong = MaxList( TrailLong, Close - ATRMult * ATRv ) ;
    Sell ( "OrLX" ) next bar at TrailLong stop ;
end ;

{ ---- ATR chandelier trailing stop: short ---- }
if MarketPosition = -1 then
begin
    if BarsSinceEntry = 0 then
        TrailShort = EntryPrice + ATRMult * ATRv
    else
        TrailShort = MinList( TrailShort, Close + ATRMult * ATRv ) ;
    BuyToCover ( "OrSX" ) next bar at TrailShort stop ;
end ;
