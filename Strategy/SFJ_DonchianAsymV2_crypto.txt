{===============================================================================
  SFJ_DonchianAsymV2_crypto
  -------------------------------------------------------------------------------
  PowerLanguage (EasyLanguage / MultiCharts64) trading strategy for crypto.
  Long + Short. ASYMMETRIC Donchian channel breakout + ReentryBars cooldown
  + ATR chandelier trail.

  SYNTHESIS of the two levers the research proved on crypto, on top of the only
  OOS-robust engine (Donchian: rolling channel + prior-edge STOP + wide ATR trail):
    1. ASYMMETRIC long/short channel lengths (SFJ_DonchianAsym showed ONLY the
       asymmetric regime is profitable -- a short LongLen for aggressive long
       entries + a very long ShortLen that suppresses bad shorts on an
       up-trending asset; symmetric Donchian lost on every combo).
    2. ReentryBars COOLDOWN -- the SFJ_DonchianATR_v2 anti-chop lever that
       actually mattered on BNB (its TrendLen MA filter went to floor / didn't
       help; the win came from the WIDE trail + the flat-only re-entry cooldown).
       After any exit the strategy must wait ReentryBars bars flat before a new
       entry, killing the immediate re-entry whipsaw that sinks bare Donchian.
  Bare asymmetric Donchian only scraped ~$5.7K (no cooldown); v2's cooldown is
  what lifted plain Donchian from all-negative to the strongest self-authored
  result. Combining both is the best shot at IS-strong AND OOS-robust.
    - Long  : flat & cooldown done -> Buy       next bar at Highest(High,LongLen)[1] STOP
    - Short : flat & cooldown done -> SellShort  next bar at Lowest(Low,ShortLen)[1]  STOP
    - Exit  : ATR(14) chandelier trailing stop (the decisive exit). Flat-only.
  Sizing: _Crypto1MUSD = Round(1,000,000 / Close, 0) ~ $1M notional per trade.

  Inputs (all optimizable; names match the optimizer's ParamAxis exactly):
    LongLen     - Donchian channel lookback for LONG breakouts (bars)
    ShortLen    - Donchian channel lookback for SHORT breakouts (bars)
    ATRMult     - trailing-stop distance in ATR(14) multiples (wider = looser)
    ReentryBars - bars to stay flat after an exit before a new entry is allowed

  Logic:
    flatBars counts consecutive flat bars (reset to 0 while in a position)
    HHl = Highest(High, LongLen)[1] ;  LLs = Lowest(Low, ShortLen)[1]   (prior-bar)
    Long  entry : flat & flatBars >= ReentryBars -> Buy       next bar at HHl STOP
    Short entry : flat & flatBars >= ReentryBars -> SellShort  next bar at LLs STOP
    Exit  : ATR chandelier trail (ATRMult * ATR(14)) under/over running extreme

  Defaults: LongLen=20, ShortLen=20, ATRMult=3.0, ReentryBars=0
  Author: generated for Tim / MultichartAI, 2026-06-24
===============================================================================}

inputs:
    LongLen( 20 ),
    ShortLen( 20 ),
    ATRMult( 3.0 ),
    ReentryBars( 0 ) ;

variables:
    _Crypto1MUSD( 0 ),
    ATRv( 0 ),
    HHl( 0 ),
    LLs( 0 ),
    flatBars( 0 ),
    TrailLong( 0 ),
    TrailShort( 0 ) ;

{ ---- position size: ~$1M notional per trade, price-normalized ---- }
_Crypto1MUSD = Round( 1000000 / Close, 0 ) ;
if _Crypto1MUSD < 1 then
    _Crypto1MUSD = 1 ;

ATRv = AvgTrueRange( 14 ) ;

{ ---- flat-cooldown counter: consecutive bars with no open position ---- }
if MarketPosition = 0 then
    flatBars = flatBars + 1
else
    flatBars = 0 ;

{ ---- asymmetric rolling Donchian channels of COMPLETED bars (non-lagged) ---- }
HHl = Highest( High, LongLen )[1] ;    { long breakout level }
LLs = Lowest ( Low , ShortLen )[1] ;   { short breakout level }

{ ---- entries: rolling-channel STOP at the edge, only when flat + cooldown done ---- }
if MarketPosition = 0 and flatBars >= ReentryBars then
begin
    Buy ( "DvLE" ) _Crypto1MUSD contract next bar at HHl stop ;
    SellShort ( "DvSE" ) _Crypto1MUSD contract next bar at LLs stop ;
end ;

{ ---- ATR chandelier trailing stop: long ---- }
if MarketPosition = 1 then
begin
    if BarsSinceEntry = 0 then
        TrailLong = EntryPrice - ATRMult * ATRv
    else
        TrailLong = MaxList( TrailLong, Close - ATRMult * ATRv ) ;
    Sell ( "DvLX" ) next bar at TrailLong stop ;
end ;

{ ---- ATR chandelier trailing stop: short ---- }
if MarketPosition = -1 then
begin
    if BarsSinceEntry = 0 then
        TrailShort = EntryPrice + ATRMult * ATRv
    else
        TrailShort = MinList( TrailShort, Close + ATRMult * ATRv ) ;
    BuyToCover ( "DvSX" ) next bar at TrailShort stop ;
end ;
