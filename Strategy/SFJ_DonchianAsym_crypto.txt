{===============================================================================
  SFJ_DonchianAsym_crypto
  -------------------------------------------------------------------------------
  PowerLanguage (EasyLanguage / MultiCharts64) trading strategy for crypto.
  Long + Short. ASYMMETRIC Donchian channel breakout + ATR chandelier trail.

  Refinement of the ONLY proven OOS-robust archetype. The research established
  that Donchian's edge is the combination, kept here UNCHANGED:
    (1) a ROLLING extreme channel (Highest/Lowest of N completed bars) -- a
        LAGGED confirmed pivot (SFJ_FractalBreakout) lost on every combo;
    (2) an intrabar STOP fill AT the channel edge -- a close-confirmed MARKET
        entry (SFJ_ChannelClose) was near-dead (enters late at a worse price);
    (3) a WIDE ATR chandelier trailing exit (the decisive lever).
  The ONE new lever is the one repeatedly shown to matter on crypto: SEPARATE
  long vs short channel lengths. Crypto is asymmetric (bull legs differ from bear
  legs); CT, HUNTER2 and XtremeStop all converged on asymmetric long/short
  parameters. A single Donchian length forces the same breakout sensitivity both
  ways; LongLen / ShortLen let the long and short sides tune independently while
  preserving the proven rolling-channel + intrabar-STOP + wide-ATR-trail engine.
    - Long  : flat -> Buy       next bar at Highest(High,LongLen)[1]  STOP
    - Short : flat -> SellShort  next bar at Lowest(Low,ShortLen)[1]   STOP
    - Exit  : ATR(14) chandelier trailing stop. Flat-only (no always-in flip).
  Sizing: _Crypto1MUSD = Round(1,000,000 / Close, 0) ~ $1M notional per trade.

  Inputs (all optimizable; names match the optimizer's ParamAxis exactly):
    LongLen  - Donchian channel lookback for LONG breakouts (bars)
    ShortLen - Donchian channel lookback for SHORT breakouts (bars)
    ATRMult  - trailing-stop distance in ATR(14) multiples (wider = looser stop)

  Logic:
    HHl = Highest(High, LongLen)[1] ;  LLs = Lowest(Low, ShortLen)[1]   (prior-bar)
    Long  entry : flat -> Buy       next bar at HHl STOP
    Short entry : flat -> SellShort  next bar at LLs STOP
    Exit  : ATR chandelier trail (ATRMult * ATR(14)) under/over running extreme

  Defaults: LongLen=20, ShortLen=20, ATRMult=3.0
  Author: generated for Tim / MultichartAI, 2026-06-24
===============================================================================}

inputs:
    LongLen( 20 ),
    ShortLen( 20 ),
    ATRMult( 3.0 ) ;

variables:
    _Crypto1MUSD( 0 ),
    ATRv( 0 ),
    HHl( 0 ),
    LLs( 0 ),
    TrailLong( 0 ),
    TrailShort( 0 ) ;

{ ---- position size: ~$1M notional per trade, price-normalized ---- }
_Crypto1MUSD = Round( 1000000 / Close, 0 ) ;
if _Crypto1MUSD < 1 then
    _Crypto1MUSD = 1 ;

ATRv = AvgTrueRange( 14 ) ;

{ ---- asymmetric rolling Donchian channels of COMPLETED bars (non-lagged) ---- }
HHl = Highest( High, LongLen )[1] ;    { long breakout level }
LLs = Lowest ( Low , ShortLen )[1] ;   { short breakout level }

{ ---- entries: rolling-channel STOP orders at the edge, only when flat ---- }
if MarketPosition = 0 then
begin
    Buy ( "DaLE" ) _Crypto1MUSD contract next bar at HHl stop ;
    SellShort ( "DaSE" ) _Crypto1MUSD contract next bar at LLs stop ;
end ;

{ ---- ATR chandelier trailing stop: long ---- }
if MarketPosition = 1 then
begin
    if BarsSinceEntry = 0 then
        TrailLong = EntryPrice - ATRMult * ATRv
    else
        TrailLong = MaxList( TrailLong, Close - ATRMult * ATRv ) ;
    Sell ( "DaLX" ) next bar at TrailLong stop ;
end ;

{ ---- ATR chandelier trailing stop: short ---- }
if MarketPosition = -1 then
begin
    if BarsSinceEntry = 0 then
        TrailShort = EntryPrice + ATRMult * ATRv
    else
        TrailShort = MinList( TrailShort, Close + ATRMult * ATRv ) ;
    BuyToCover ( "DaSX" ) next bar at TrailShort stop ;
end ;
