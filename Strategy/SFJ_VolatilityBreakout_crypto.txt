{===============================================================================
  SFJ_VolatilityBreakout_crypto
  -------------------------------------------------------------------------------
  PowerLanguage (EasyLanguage / MultiCharts64) trading strategy for crypto.
  Long + Short. Volatility-breakout STOP entry (Larry-Williams style) + ATR
  chandelier trailing exit.

  NEW entry archetype vs the rest of the Strategy folder. The crypto research
  concluded the EXIT is decisive (a WIDE ATR chandelier trail is what makes
  Donchian both IS-strong AND OOS-robust), and that the WINNING family is
  "breakout entry + wide ATR trail". This system keeps that exit but uses a
  DIFFERENT, volatility-scaled entry trigger:
    - Donchian enters at the N-bar channel extreme; Keltner at an EMA +/- band.
    - Here the breakout level is anchored to the CURRENT price plus a
      volatility-scaled buffer:  BuyLevel = Close + EntryMult * ATR.
      Price must thrust EntryMult ATRs beyond the last close to trigger -- a
      volatility-normalized breakout that self-adapts to regime (the buffer
      widens in high-vol, tightens in low-vol). The buffer is the anti-false-
      breakout lever (larger EntryMult = fewer, more decisive breakouts).
    - Flat-only entry (no always-in flipping); exit = ATR(ATRLen) chandelier
      trail at TrailMult (kept SEPARATE + typically WIDER than the entry buffer,
      since the research showed wide trails win).
  Sizing: _Crypto1MUSD = Round(1,000,000 / Close, 0) ~ $1M notional per trade.

  Inputs (all optimizable; names match the optimizer's ParamAxis exactly):
    ATRLen    - ATR period used for BOTH the entry buffer and the trail
    EntryMult - breakout-buffer distance in ATR multiples (entry trigger)
    TrailMult - trailing-stop distance in ATR multiples (exit; usually > EntryMult)

  Logic:
    ATRv = AvgTrueRange(ATRLen)
    BuyLevel  = Close + EntryMult * ATRv ;  SellLevel = Close - EntryMult * ATRv
    Long  entry : flat -> Buy       next bar at BuyLevel  STOP
    Short entry : flat -> SellShort  next bar at SellLevel STOP
    Exit  : ATR chandelier trail (TrailMult * ATRv) under/over running extreme

  Defaults: ATRLen=14, EntryMult=1.0, TrailMult=3.0
  Author: generated for Tim / MultichartAI, 2026-06-24
===============================================================================}

inputs:
    ATRLen( 14 ),
    EntryMult( 1.0 ),
    TrailMult( 3.0 ) ;

variables:
    _Crypto1MUSD( 0 ),
    ATRv( 0 ),
    BuyLevel( 0 ),
    SellLevel( 0 ),
    TrailLong( 0 ),
    TrailShort( 0 ) ;

{ ---- position size: ~$1M notional per trade, price-normalized ---- }
_Crypto1MUSD = Round( 1000000 / Close, 0 ) ;
if _Crypto1MUSD < 1 then
    _Crypto1MUSD = 1 ;

{ ---- volatility-scaled breakout levels (anchored to the completed bar) ---- }
ATRv      = AvgTrueRange( ATRLen ) ;
BuyLevel  = Close + EntryMult * ATRv ;
SellLevel = Close - EntryMult * ATRv ;

{ ---- entries: volatility-breakout STOP orders, only when flat ---- }
if MarketPosition = 0 then
begin
    Buy ( "VbLE" ) _Crypto1MUSD contract next bar at BuyLevel stop ;
    SellShort ( "VbSE" ) _Crypto1MUSD contract next bar at SellLevel stop ;
end ;

{ ---- ATR chandelier trailing stop: long ---- }
if MarketPosition = 1 then
begin
    if BarsSinceEntry = 0 then
        TrailLong = EntryPrice - TrailMult * ATRv
    else
        TrailLong = MaxList( TrailLong, Close - TrailMult * ATRv ) ;
    Sell ( "VbLX" ) next bar at TrailLong stop ;
end ;

{ ---- ATR chandelier trailing stop: short ---- }
if MarketPosition = -1 then
begin
    if BarsSinceEntry = 0 then
        TrailShort = EntryPrice + TrailMult * ATRv
    else
        TrailShort = MinList( TrailShort, Close + TrailMult * ATRv ) ;
    BuyToCover ( "VbSX" ) next bar at TrailShort stop ;
end ;
